Quick Start

Prerequisites

  • n8n instance (cloud or self-hosted)
  • Node.js 18+
  • n8nac CLI (npm install)

Install

# Clone and set up
git clone https://github.com/YOUR-USER/YOUR-PROJECT.git
cd YOUR-PROJECT
npm install
npx n8nac init

# Scaffold your first workflow
npm run new-workflow -- agents/01-my-agent "My Agent"

# Push to n8n
npx n8nac push my-agent.workflow.ts
